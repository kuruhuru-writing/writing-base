# Запрос: составить план следующей главы

На основе файлов романа составь гибкий план следующей главы.

Важно: не делай слишком подробный жёсткий план. Сохрани пространство для импровизации.

План должен включать:

- функцию главы;
- ближайший маяк;
- 2–4 возможные сцены;
- цель героя в каждой сцене;
- конфликт;
- возможное осложнение;
- реакцию героя, если она нужна;
- какие promises/questions могут обновиться.

Формат:

```markdown
# План главы

## Функция главы

## Связь с ближайшим маяком

## Сцена 1

## Сцена 2

## Сцена 3

## Возможная финальная нота главы

## Что оставить открытым для импровизации

## Что нужно обновить после главы
```
