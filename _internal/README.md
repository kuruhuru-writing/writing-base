# _internal

Эта папка предназначена для технических и агентских инструкций.

Сюда входят:

- протокол работы ИИ-агента;
- карта файлов;
- спецификация пакета контекста;
- критерии качества;
- соглашения по именованию;
- Git-процесс;
- рекомендации по Obsidian и Longform.

Файлы в `_internal` не являются художественным текстом и не должны попадать в рукопись.
