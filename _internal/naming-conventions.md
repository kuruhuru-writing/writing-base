# Соглашения по именованию

## Репозитории

```text
writing-base
novel-template
novel-<working-title>
writing-tools
```

## Сцены

```text
001-opening.md
002-reaction.md
003-first-complication.md
```

Правила:

- номер с ведущими нулями;
- короткое английское или транслитерированное название;
- без пробелов;
- `kebab-case`.

## Персонажи

```text
characters/protagonist.md
characters/antagonist.md
characters/maria-sokolova.md
```

## Журнал

```text
03-journal/daily/2026-06-13.md
03-journal/writing-log.md
03-journal/idea-inbox.md
```

## Решения

```text
01-planning/decisions.md
01-planning/decisions/2026-06-13-final-tone.md
```

Если решений станет много, можно вынести их в отдельную папку.
