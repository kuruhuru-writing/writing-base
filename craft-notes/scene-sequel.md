# Scene / Sequel

## Scene

```text
Goal → Conflict → Disaster/Complication
```

Герой пытается добиться цели и сталкивается с сопротивлением.

## Sequel

```text
Reaction → Dilemma → Decision
```

Герой переживает последствия и принимает решение.

## Практический цикл

```text
Решение героя → новая цель → сцена действия → осложнение → реакция → новое решение
```

Это позволяет строить роман как цепь последствий, а не как набор эпизодов.
