# Чек-лист главы

## Структура

- [ ] Глава содержит одну или несколько сцен с ясной функцией.
- [ ] В начале главы есть крючок или напряжение.
- [ ] В конце главы есть изменение ситуации.
- [ ] Глава не состоит только из экспозиции.

## Динамика

- [ ] В главе есть движение цели, конфликта или знания.
- [ ] Эмоциональная интенсивность меняется, а не стоит на одном уровне.
- [ ] Есть причина читать дальше.

## Персонаж

- [ ] Герой в главе чего-то хочет.
- [ ] Герой сталкивается с сопротивлением.
- [ ] Герой делает выбор, узнаёт новое или платит цену.

## Непрерывность

- [ ] Глава не противоречит уже принятым решениям.
- [ ] Обещания читателю учтены.
- [ ] Открытые вопросы не забыты.
- [ ] Если добавлены новые линии, они зафиксированы.
