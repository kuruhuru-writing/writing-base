# Чек-лист непрерывности

## Сюжет

- [ ] События следуют по причинности, а не просто по очереди.
- [ ] Нет случайного поворота без подготовки или последствий.
- [ ] Активные линии не исчезли без причины.

## Персонажи

- [ ] Мотивация персонажей не противоречит их карточкам.
- [ ] Отношения меняются с учётом предыдущих сцен.
- [ ] Персонажи помнят значимые события.

## Мир и факты

- [ ] Правила мира соблюдены.
- [ ] Временная линия не ломается.
- [ ] Локации и расстояния не противоречат друг другу.

## Обещания

- [ ] Все открытые обещания есть в `promises.md`.
- [ ] Есть понимание, какие обещания должны закрыться позже.
- [ ] Нет ложных обещаний, которые текст подал случайно.

## Вопросы

- [ ] Все значимые вопросы есть в `questions.md`.
- [ ] Понятно, какие вопросы остаются открытыми намеренно.
