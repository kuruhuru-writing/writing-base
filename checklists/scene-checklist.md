# Чек-лист сцены

## Базовая функция

- [ ] У сцены есть понятная функция в романе.
- [ ] Понятно, кто является героем сцены.
- [ ] У героя есть конкретная ближайшая цель.
- [ ] Есть сопротивление, препятствие или конфликт.
- [ ] В конце сцены ситуация меняется.
- [ ] Итог сцены рождает последствия.

## По Суэйну

Для сцены действия:

- [ ] Цель ясна.
- [ ] Конфликт активен.
- [ ] Осложнение достаточно сильное.

Для сцены реакции:

- [ ] Есть эмоциональная реакция.
- [ ] Есть осмысление.
- [ ] Есть дилемма.
- [ ] Есть решение.
- [ ] Решение рождает новую цель.

## Причинность

- [ ] Сцена вытекает из предыдущих событий.
- [ ] Сцена не выглядит случайной вставкой.
- [ ] Следующая сцена может логично родиться из этой.

## Читательский интерес

- [ ] В сцене есть напряжение.
- [ ] Есть вопрос, на который хочется получить ответ.
- [ ] Герой раскрывается через действие, выбор или реакцию.
- [ ] Сцена не перегружена объяснениями.

## После сцены

- [ ] Обновлены promises.md, если появились обещания.
- [ ] Обновлены questions.md, если появились вопросы.
- [ ] Обновлены decisions.md, если принято решение.
- [ ] Обновлён writing-log.md.
