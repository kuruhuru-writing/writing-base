# Чек-лист редактуры

## Первый проход: смысл и причинность

- [ ] У каждого крупного события есть причина.
- [ ] Каждое крупное событие имеет последствия.
- [ ] Главный герой не исчезает из собственной истории.
- [ ] Нет сцен, которые ничего не меняют.

## Второй проход: сцены

- [ ] У каждой сцены есть цель, конфликт и итог.
- [ ] После сильных ударов есть реакция или её осознанное отсутствие.
- [ ] Сцены не повторяют одну и ту же функцию.

## Третий проход: читатель

- [ ] Начало держит внимание.
- [ ] Напряжение развивается.
- [ ] Вопросы дозируются.
- [ ] Развязки подготовлены.

## Четвёртый проход: язык

- [ ] Убраны лишние объяснения.
- [ ] Диалоги не пересказывают очевидное.
- [ ] Описания работают на атмосферу, конфликт или характер.
- [ ] Ритм фраз соответствует сцене.
