# Методология этого романа

Этот файл — снимок методологии, применяемой к конкретному роману.

Базовая формула:

```text
Компас → Маяки → Сцены → Реакции → Журнал → Проверка → Пересборка
```

## Как писать

1. Держать направление в `01-planning/compass.md`.
2. Держать крупные ориентиры в `01-planning/beacons.md`.
3. Планировать только ближайшие 1–3 сцены.
4. Писать сцены через цель, конфликт и осложнение.
5. После сильных событий давать реакцию: эмоция, дилемма, решение.
6. Обновлять журнал и состояние проекта.
7. Разрешать импровизацию, если она усиливает роман.

## Главный принцип

Не нужно заранее знать весь путь. Нужно понимать, почему следующая сцена неизбежно рождается из текущего состояния.
