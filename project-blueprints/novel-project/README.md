# Novel Project

Это заготовка репозитория конкретного романа.

## Структура

```text
00-method/      # Методология, применяемая к этому роману
01-planning/    # Компас, маяки, обещания, вопросы, решения
02-manuscript/  # Сцены и рукопись
03-journal/     # Журнал работы
04-output/      # Собранные версии рукописи
05-archive/     # Старые и удалённые материалы
```

## Первый запуск

1. Заполни `01-planning/compass.md`.
2. Заполни `01-planning/beacons.md`.
3. Создай карточку главного героя в `01-planning/characters/`.
4. Начни первую сцену в `02-manuscript/scenes/001-opening.md`.
5. После письма обнови `03-journal/writing-log.md`.
6. Сделай commit.
