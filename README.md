# Writing Base

`writing-base` — это общий репозиторий писательской системы: методология, шаблоны, чек-листы, заметки по ремеслу и инструкции для ИИ-агентов.

Этот репозиторий **не предназначен для хранения рукописей конкретных романов**. Конкретный роман лучше держать в отдельном репозитории, созданном по заготовке из `project-blueprints/novel-project`.

## Главная идея

Метод построен вокруг гибридного письма:

> Автор заранее задаёт направление, крупные опорные точки и правила движения сцен, но оставляет пространство для импровизации и открытия сюжета в процессе.

Короткая формула процесса:

```text
Компас → Маяки → Ближайшие сцены → Сцена по Суэйну → Реакция → Журнал → Проверка → Следующая сцена
```

## Структура репозитория

```text
writing-base/
  README.md
  GETTING_STARTED.md
  AGENTS.md
  .gitignore

  methodology/         # Основная методология письма
  templates/           # Шаблоны заметок, сцен, персонажей, журналов
  checklists/          # Проверочные списки
  craft-notes/         # Заметки по писательскому ремеслу
  agent-prompts/       # Готовые запросы к ИИ-агенту
  project-blueprints/  # Заготовки будущих репозиториев романов
  _internal/           # Технические инструкции для ИИ-агентов и процесса
```

## Как использовать

1. Открой этот репозиторий как Obsidian vault или просто как папку Markdown-файлов.
2. Читай `GETTING_STARTED.md`.
3. Развивай методологию в `methodology/`.
4. Храни универсальные шаблоны в `templates/`.
5. Для нового романа скопируй `project-blueprints/novel-project/` в отдельный репозиторий.
6. Для консультаций с ИИ-агентом давай ему:
   - `AGENTS.md`;
   - `_internal/agent-operating-protocol.md`;
   - методологию из `methodology/`;
   - файлы конкретного романа.

## Что здесь не хранить

Не стоит хранить в `writing-base`:

- полный текст конкретного романа;
- приватные черновики романов;
- большие экспортированные `.docx`/`.pdf`, если они относятся к конкретному проекту;
- хаотичные идеи без привязки к методологии или шаблонам.

Для этого создаётся отдельный репозиторий романа.
